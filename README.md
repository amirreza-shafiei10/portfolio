## Features

- **Bilingual (FR / EN)** — Full French/English toggle with instant switching and `localStorage` persistence
- **Dark / Light Mode** — Default dark theme, toggleable with preference saved across sessions
- **Typing Animation** — Hero subtitle cycles through roles (Web Developer / CS Student / App Builder)
- **Particle Background** — Subtle animated mesh/dot canvas in the hero section
- **Scroll Reveal** — Sections animate in as you scroll down
- **Featured Project** — CalTrack AI nutrition tracker highlighted as the lead project with live demo
- **Tech Stack Badges** — Skills displayed in grouped badge system (Frontend / Languages / Tools)
- **Education Timeline** — Visual timeline of academic background
- **Inline Form Feedback** — Contact form validation with styled inline messages (no alert() popups)
- **Back to Top Button** — Appears on scroll, smooth-scrolls to top
- **Responsive Design** — Fully mobile-friendly via Bootstrap 5 grid
- **SEO & Social Ready** — Meta description, Open Graph and Twitter Card tags

## Getting Started

No build step required — open `index.html` directly in a browser, or serve with any static file server:

```bash

python -m http.server 8000

# Using Node.js (npx)
npx serve .
```
